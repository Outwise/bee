# bees.direct

Bees. Direct to you.

[bees.direct](http://bees.direct)

## Play application

### Build

    $ sbt play/dist
    $ sbt play/publishLocal

### Run

    $ sbt play/run

### Docker run

    $ docker run -it jrouly/bees-direct-play:latest

Also available on [hub.docker.com](https://hub.docker.com/r/jrouly/bees-direct-play/).


## Lambda function

### Build

    $ sbt lambda/assembly
    $ serverless deploy
