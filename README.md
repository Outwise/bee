# bees.direct

Bees. Direct to you.

[bees.direct](http://bees.direct)

## Building jars

    $ sbt dist

## Docker

    $ docker run -it jrouly/bees-direct-play:latest

Available at [hub.docker.com](https://hub.docker.com/r/jrouly/bees-direct-play/).

### Publishing locally

    $ sbt bees-direct-play/docker:publishLocal
